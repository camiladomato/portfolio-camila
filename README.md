## Secciones

- **Hero** — Presentación con badge de disponibilidad y tarjeta visual de perfil
- **Sobre mí** — Stack tecnológico, formación (UTN + MindHub) y experiencia como mentora
- **Servicios** — Desarrollo web, branding e identidad visual, digitalización de negocios
- **Proyectos** — Proyectos reales con links al repositorio
- **Contacto** — Formulario con integración directa a WhatsApp

## Stack

- [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/)
- Animaciones con `IntersectionObserver` (sin dependencias extra)

## Correr en local

```bash
npm install
npm run dev
```

## Build para producción

```bash
npm run build
```
